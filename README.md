# Gestor de Tareas

Aplicación web desarrollada con React para la gestión y organización de tareas. Permite crear, editar, eliminar y clasificar tareas por categorías y prioridades, además de ofrecer una experiencia interactiva mediante animaciones, búsqueda en tiempo real y funcionalidad de arrastrar y soltar.

## Características

*  Crear tareas
*  Editar tareas
*  Eliminar tareas
*  Marcar tareas como completadas
*  Organización por categorías
*  Sistema de prioridades
*  Búsqueda de tareas en tiempo real
*  Filtro por categorías
*  Modo oscuro
*  Persistencia de datos con Local Storage
*  Drag & Drop entre categorías
*  Animaciones al completar y eliminar tareas
*  Diseño responsive para dispositivos móviles

## Tecnologías utilizadas

* React
* JavaScript (ES6+)
* Vite
* CSS3
* Local Storage API
* DnD Kit

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/TU-USUARIO/task-manager.git
```

Ingresa al proyecto:

```bash
cd task-manager
```

Instala las dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm run dev
```
